# Fitness Analytics Dashboard
## Link: https://gymandcalorieprogress.streamlit.app/

A Python web app for tracking workouts, nutrition, and body weight, with interactive charts and data-driven insights.

## Features

- Log meals, workouts, and body weight
- Visualize trends 
- Muscle group distribution
- Filter recent data
- Edit and delete entries
- Insight system to highlight patterns:
  - Low protein intake
  - Calorie surplus/deficit
  - Workout imbalances

## Tools used

- Python
- Streamlit
- Pandas
- Plotly
